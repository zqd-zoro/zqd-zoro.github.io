CentOS 7下编译安装 apache

Posted on 2019-07-30

CentOS 7下编译安装 apache

1.在每安装一个服务都要养成查看是否安装，如果安装则需要卸载
rpm -qa | grep httpd
安装需要的组件
yum -y install libxml2-devel
yum -y install openssl-devel
yum -y install expat-devel
yum -y install gcc gcc-c++
2.安装apache2.4.23 （是编译安装那么就会需要用到源码包）在安装apache2.4.23需要的源码包如下:
httpd-2.4.23.tar.gz( wget http://httpd.apache.org/download.cgi#apache24 )
apr-1.5.2.tar.gz ( wget http://mirrors.hust.edu.cn/apache//apr/apr-1.5.2.tar.gz)
apr-util-1.5.4.tar.gz (wget http://mirrors.hust.edu.cn/apache//apr/apr-util-1.5.4.tar.gz)
zlib-1.2.8.tar.gz(wget http://www.zlib.net/fossils/zlib-1.2.8.tar.gz)
pcre-8.39.tar.gz(wget https://ftp.pcre.org/pub/pcre/pcre-8.39.tar.gz)
然后解压到 /data/server/下

然后cd到相对应的路径下进行编译安装
(执行1../configure,2.make,3.make install ,三个命令来编译安装这个东西)
然后检查是否安装了httpd_devel，如果没有安装，执行yum install httpd-devel,进行安装，如果不安装的话，后期不会生成.so的扩展文件。
. /configure –prefix=/data/server

然后用make make install 分两步进行编译安装 用echo $? 查看下错误

apr-util需要依赖 apr 所以要这样执行 –with-apr=/data/server/apr

然后用make make install 分两步进行编译安装 用echo $? 查看下错误 同上

同样 make make install 同上

./configure –prefix=/data/server/apache-2.4.39 –with-apr=/data/server/apr –with-apr-util=/data/server/apr-util –with-pcre=/data/server/pcre –with-zlib=/data/server/zlib –enable-so
然后到bin下启动 然后访问IP就安装好apache了

关闭防火墙 systemctl stop firewalld
1 下载php5.6安装包到/usr/loca/src
解压到 /data/server/下 改名php56
2 cd /data/server/apache24/conf
vim httpd.conf 修改配置文件
(在loadmodule后添加 loadmodule php5_modules/libphp5.so
在Addtype application/x-gizp.gz.tgz后面添加 AddType application/x-httpd-php //.php 后面有个空格
3
4编译安装php
PHP编译参数：
./configure –prefix=/data/server/php56 –with-iconv-dir –with-freetype-dir –with-jpeg-dir –with-png-dir –with-zlib –with-libxml-dir –enable-simplexml –enable-xml –disable-rpath –enable-bcmath –enable-soap –enable-zip –with-curl –enable-fpm –with-fpm-user=www –with-fpm-group=www –enable-mbstring –enable-sockets –with-gd –with-openssl –with-mhash –enable-opcache –disable-fileinfo –with-apxs2=/data/server/apache24/bin/apxs
5 到这个目录下启动 关闭防火墙

报错端口冲突 修改端口
cd /data/server/apache24/conf
vim httpd.conf
找到Listen 80这一行，修改为
Listen 8999 (我这修改成8999)
